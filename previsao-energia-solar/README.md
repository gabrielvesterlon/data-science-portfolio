# Previsão de Geração de Energia Solar com Machine Learning

Este projeto utiliza inteligência artificial para prever a produção de energia solar com base em variáveis climáticas. Desenvolvido como parte do meu portfólio de Ciência de Dados, ele demonstra o fluxo completo desde o tratamento de dados até a visualização de resultados.

## Tecnologias e Ferramentas
* **Linguagem:** Python
* **Manipulação de Dados:** Pandas e SQL (SQLite)
* **Machine Learning:** Scikit-learn (Regressão Linear)
* **Visualização:** Power BI e Matplotlib

## O Projeto
O fluxo de trabalho consistiu em:
1. **Geração de Dados Sintéticos:** Criação de um dataset simulando condições climáticas e produção de energia, estruturado via Python e armazenado em SQLite.
2. **Processamento:** Limpeza e tratamento dos dados com Python.
3. **Modelagem:** Treinamento de um modelo de regressão para prever a geração de energia.
4. **Análise de Resultados:** O modelo alcançou um **$R^2$ de aproximadamente 73%**, demonstrando uma boa capacidade preditiva para o cenário proposto.
5. **Dashboard:** Criação de um painel interativo no Power BI para visualização dos indicadores.
